Project Hydra

Homelab Kubernetes Cluster Documentation (k3s + NixOS + PiOS Lite)

Overview

Project Hydra is a distributed homelab Kubernetes environment built using k3s, running across NixOS-based machines and Raspberry Pi nodes. It is a lightweight, reproducible, and highly available cluster for learning, experimentation, and self-hosted services.

Cluster Architecture

  • Control Plane: hydra-1 (primary k3s server)
  • Worker Nodes: hydra-4 + Raspberry Pi nodes (PiOS Lite)
  • Networking: Flat LAN cluster network
  • Container Runtime: containerd (via k3s)

Current Nodes

  • hydra-1 → Kubernetes server / control plane
  • hydra-4 → Worker node (general workloads)
  • Pi nodes (hydra-3 & hydra-4) → Lightweight compute / future expansion
  • hydra-5 Worker node (general workloads)

Namespaces

The cluster is organized using Kubernetes namespaces for isolation and structure.

  • default → general workloads
  • kube-system → system components
  • kube-public / kube-node-lease
  • hercules → application namespace (monitoring workloads)

Observability

Monitoring is being introduced using Grafana-based tooling within the cluster.

  • Grafana deployed internally for metrics visualization
  • Accessible over LAN (not yet publicly exposed)
  • Planned: Prometheus integration for full cluster metrics

Storage Layer

Persistent storage is planned using Longhorn for distributed block storage across nodes.

  • Target: replicate storage across hydra-1 and hydra-4
  • Goal: fault tolerance for stateful workloads
  • Current status: not yet deployed

Tooling & Management

  • kubectl for cluster management
  • NixOS for reproducible system configuration
  • k3s for lightweight Kubernetes distribution
  • SSH-based admin access from personal workstation

Current Goals

  • Deploy Longhorn across capable nodes
  • Standardize Helm usage for reproducible deployments
  • Improve Git-based versioning of Kubernetes manifests
  • Expose selected services securely over LAN (and later WAN if desired)
  • Add CI/CD pipeline for cluster updates
